PROJECTS

MonitorInputSwitcher

This project controls the inputs of a monitor using an Arduino. When the Arduino is connected to a computer, it runs the installed program to communicate to a serial monitor. The MonitorInputSwitcher program then reads the serial monitor to determine which command it should run.
